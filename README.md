### chibi-Redis

A minimalist, educational Redis clone written in C++17.  

'chibi-Redis' is a lightweight, toy implementation of the Redis in-memory data structure store.  
This projet is built entirely from scratch to understand how Redis works.

- Language: C++17
- Build System: CMake + Ninja
- OS: Ubuntu-24.04

The support commands are following.

- PING
- ECHO
- SET
- GET
- EXISTS
- DEL
- INCR
- DECR
- LPUSH
- RPOP

The test directory includes test script generated by AI.
